# OmniQR: Future Roadmap (ROADMAP.md)

Strategic vision for the evolution of the OmniQR Enterprise Suite.

## 🏁 Phase 4: Social & Identity Integration (Short Term)
- **Unified Identity (vCard)**: A specialized QR type for professional contact sharing with real-time update capabilities.
- **Social Connect Hubs**: Link-aggregation pages ("Bio" pages) hosted directly on the Smart Asset engine.
- **Batch Asset Generation**: Ability to generate 100+ QR codes from a CSV/Excel file for mass inventory tracking.

## 📈 Phase 5: Deep Analytics Dashboard (Mid Term)
- **Advanced Scan Metrics**: Tracking scans by geography (country/city) and device type (iOS/Android).
- **A/B Testing Redirection**: Dynamically splitting QR traffic between two different target URLs to test marketing effectiveness.
- **Export Engine**: Support for SVG and High-Resolution EPS formats for professional printing.

## 🛡️ Phase 6: Enterprise Security Plus (Long Term)
- **RBAC (Role-Based Access Control)**: Permission levels for Admin, Editor, and Viewer roles in the console.
- **Password Protected QRs**: Require a PIN or password before a user can access a Smart Cloud Asset.
- **API for Developers**: A REST API to allow other enterprise apps to generate and manage assets programmatically.

---

## 🛠️ Upcoming Technical Debt / Refinement
- **Image Cropper**: Integrated cropping tool for user brand logos.
- **Service Worker**: PWA support for offline access to the QR Library.
- **Asset Expiration**: Ability to set a "self-destruct" date for Smart Cloud Assets.
